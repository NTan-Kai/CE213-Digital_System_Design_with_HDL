# LAB05 – Simple MIPS Datapath

This lab implements a simple MIPS-style datapath by integrating register file, sign extension, ALU, data memory, and multiplexer modules.

## Contents

- `src/`: Datapath source files and required modules.
- `tb/Tb_Datapath.v`: Datapath testbench.

## Tools

- Verilog HDL
- ModelSim
