# CE213 Source Structure

This package contains organized Verilog HDL source files and testbenches for the CE213 repository.

## Included Labs

- LAB01: 3-bit Carry Lookahead Adder
- LAB02: Synchronous Counter with Load
- LAB03: Moore 010 FSM and SRAM Data Memory
- LAB04: 32-bit ALU and 2-to-1 Multiplexer
- LAB05: Simple MIPS Datapath
- LAB06: Control Unit
- LAB07: Simple Processor

Temporary simulation files such as `.bak`, `.mpf`, `.cr.mti`, and `vsim.wlf` were not included.
